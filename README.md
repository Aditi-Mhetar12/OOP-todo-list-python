**OOP To-Do List Application**

This project is a command-line based To-Do List application developed using Python and Object-Oriented Programming (OOP) concepts.
It allows users to efficiently manage daily tasks with features such as adding, updating, deleting, and viewing tasks.
The application uses JSON file handling to ensure that tasks are stored persistently and are not lost after the program exits.

Technologies Used

Python 3

Object-Oriented Programming (OOP)

JSON (File Handling)

Exception Handling
